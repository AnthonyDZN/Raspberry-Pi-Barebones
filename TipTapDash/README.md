## --{ SETUP }--
**r0** - GPBASE <br>
**r1** - Time<br>
**r2** - Score<br>
**r3** - Button1 (P1)<br>
**r4** - Button2 (P2)<br>


## --{ GAME START }--
**r1** - Time<br>
**r2** - Score<br>
**r5** - Mask for P1 ready<br>
**r6** - Mask for P2 ready<br>
**r7** - LED P1<br>
**r8** - LED P2 <br>
**r9** - Button Test <br>
**r10** - Button2 Test <br>
